# Project1_StockTraderApp

# Overview

The Stock Tracker App is a simple web application that allows users to track stock performance. Users can search for specific stock symbols to see their current performance or view the top 10 performing stocks. The application fetches real-time stock data from the Alpha Vantage API.

# Features

(1) Search for specific stock symbols to view their performance.

(2) Display top 10 performing stocks by default.

(3) Responsive and modern design.

(4) Real-time data fetching from the Alpha Vantage API.

# Technologies Used

HTML
CSS
JavaScript
Alpha Vantage API

# Usage

Search for a Stock:

(1) Enter a stock symbol in the input field.
(2) Click the "Search" button to fetch and display the stock data.
(3) View Top 10 Stocks: By default, the app displays the top 10 performing stocks.

# Customization

API Key:

The application uses the Alpha Vantage API. To use your own API key, replace YOUR_API_KEY in the index.js file with your actual Alpha Vantage API key.

Styling:

You can customize the look and feel of the application by modifying the style.css file.
